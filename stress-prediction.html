AI-Based Stress Prediction System

An intelligent system that monitors physiological signals to predict and manage stress levels in real-time

Project Overview

This project is about creating a smart system that can detect when a person is feeling stressed by using small sensors that measure things like heart rate and skin moisture. These sensors send the information to a tiny computer, which then uses artificial intelligence (AI) to figure out how stressed the person is‚Äîwhether they are relaxed, a little stressed, or very stressed.

Technical Implementation

Hardware Components

  • GSR Sensor: Measures stress levels by detecting changes in skin moisture caused by sweat glands and emotions using two electrodes placed on the skin
  • Heart Rate Sensor: Attached to a finger to measure pulse rate
  • ESP32 Microcontroller: Processes sensor data and handles cloud communication
  • DHT11 Sensor: Monitors environmental temperature and humidity for context

Software Architecture

  • Arduino C++: Main programming language for ESP32
  • ThingSpeak API: Cloud platform for data storage and visualization
  • ANFIS Algorithm: Adaptive Neuro-Fuzzy Inference System for stress classification
  • Web Dashboard: Real-time monitoring interface built with HTML/CSS/JavaScript

Results & Validation

The system was tested and it worked well. It was able to read heart rate and GSR (skin response) from the sensors. The data was shown live on a web dashboard. When someone was stressed, the system gave an alert. The AI model could tell if the person was calm or stressed.


üìä Performance Metrics

  • Classification Accuracy: ~90%
  • Sensitivity: ~88%
  • Specificity: ~92%
  • Response Time: <1 second

Overall, the project gave good results and proved that stress can be detected using sensors and AI in real time. The system successfully demonstrated the viability of edge-based stress monitoring with competitive performance compared to cloud-based alternatives.

Future Enhancements
